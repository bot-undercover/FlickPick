Это api позволяет генерировать ответы с помощью ИИ

Предупреждение: Получая данные с сервера - они стираются

Пример запроса:

curl -d "{'action': 'generate', 'text':'Какой лучший новогодний фильм можешь посоветовать?'}" {{ ip }}/api

Примерный ответ от сервера:

{'key': 'ключ'}

Ключ - 128 символьная строка, потребующаяся для обращения на сервер

Пример запроса на получение результата:

curl -d "{'action': 'get', 'key':'ключ'}" {{ ip }}/api

Примерный ответ от сервера:

· Если ответ еще продолжает обрабатываться или вы уже его забрали с сервера:

{'answer': 'None'}

· Если ответ уже образовался:

{'answer': 'Одним из лучших новогодних фильмов, который стоит посмотреть, является "Ирония судьбы, или С днём рождения" (1975) режиссёра Элема Климовича. Это советский фильм, который рассказывает о двух людях, которые случайно встречаются в новогоднюю ночь в отеле. Фильм полон юмора, романтики и глубоких философских размышлений о судьбе человека.'}
